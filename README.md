# Sky-Weather
SkyWeather is a modern weather application that shows real-time weather information based on user location or searched city. It provides accurate data with a clean and user-friendly interface.

# 🌤️ SkyWeather

---

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Location-based weather detection
- 🌡️ Real-time temperature updates
- 💨 Wind speed, humidity, pressure
- 📅 7-day weather forecast
- 🌙 Light/Dark mode UI

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- Weather API (OpenWeatherMap API)
- Responsive Design

---

## 💡 How It Works

SkyWeather fetches real-time weather data from an external API and displays it in a structured and easy-to-understand format for users.
